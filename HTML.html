Tags

< Voltar

As tags em HTML (HyperText Markup Language) são elementos fundamentais da linguagem que servem para estruturar e definir o conteúdo de uma página da web. Elas são usadas para identificar e descrever diferentes partes do documento, como cabeçalhos, parágrafos, imagens, links e muito mais. Elas podem ser:

  • Tag de abertura: Marca o início de um elemento. Exemplo :
    <a>
  • Tag de fechamento: Marca o fim do elemento. Exemplo: 
    </a>

Algumas tags são auto-contidas e não possuem uma tag de fechamento, elas são: Tags Auto-Fechadas.Exemplo: 

<img>

Algumas Tags Importantes

  • <strong> -> Deixa em negrito
  • <i> -> deixa em itálico
  • <p> -> Cria um parágrafo
  • <mark> -> deixa destacado
  • <blockquote> Usado para fazer:

Atributos

< Voltar

Os atributos são propriedades adicionais usadas para fornecer informações ou configurar o comportamento de elementos HTML. Eles são sempre declarados dentro da tag de abertura e consistem em um nome de atributo e um valor, separados por um sinal de igual. Em resumo elas são completmentos dos tags

Características dos Atributos

  1. Sempre na tag de abertura.

  2. Nome e valor: O nome do atributo é sensível ao padrão (não diferencia maiúsculas de minúsculas, mas é recomendável usar minúsculas por convenção). O valor geralmente é colocado entre aspas ("), embora seja opcional em alguns casos.

  3. Personalizam ou configuram a tag.

Exemplos de Atributos Comuns

  1. Atributos Gerais:
    • id: Identifica um elemento de forma única na página.
      • <div id="principal">Conteúdo principal</div>
    • class: Associa um elemento a uma ou mais classes CSS.
      • <p class="destaque">Texto destacado</p>
  2. Atributos Específicos de Elementos:
    • href: Define o endereço de destino do link.

    • target: Especifica onde abrir o link (por exemplo, em uma nova aba com _blank).
      • <a href="https://google.com" target="_blank">Visite o Google</a>
    • src: Define o caminho da imagem.

    • alt: Texto alternativo caso a imagem não seja carregada.
      • <img src="imagem.jpg" alt="Descrição da imagem">
  3. Atributos Globais:
    • style: Adiciona estilos CSS diretamente ao elemento.
      • <h1 style="color: red;">Título em vermelho</h1>
    • title: Define um texto de ajuda exibido ao passar o mouse sobre o elemento.
      • <abbr title="HyperText Markup Language">HTML</abbr>

Observações Importantes de Atributos

  • Atributos Booleanos: Alguns atributos não precisam de valor, pois sua presença já indica que estão ativados. Exemplo:
    • <input type="checkbox" checked>
  • Validade e Padrões: É importante usar os atributos corretos para cada elemento e garantir que estejam dentro dos padrões do HTML5.

Os atributos são fundamentais para tornar os elementos mais dinâmicos, personalizados e interativos!
